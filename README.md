## Request-for-Tender implementation on Celo Network

This is an implementation of a Request-for-Tender (RFT) system on Celo network. The RFT system is a decentralized application (dApp) that allows users to create and manage RFTs on the Celo network. 

### Prospective Use-cases
1. `msq.sender` can mint (upload) their tender offers to the blockchain inform of NFTs.
2. Service providers with valid celo wallets can seek for tender NFT offers and bid as per the declared timeline.
3. `msg.sender` can lookup and decide on the quality of the service provider by utilizing Utu protocol.
4. `msq.sender` can accept the best bid and award the tender to the service provider.
5. The service provider can then redeem the tender NFT for the agreed amount of celo tokens.
